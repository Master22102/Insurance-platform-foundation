This section appears on the trip detail page when TravelShield is active for this trip. It sits between the route segments section and the coverage section. Tap "View on map" for full-screen map with partner location.

What this card shows:

Partner's avatar, name, and online status (green dot = active, amber = last seen >30min, red = no signal)
Mini map placeholder showing approximate location (actual map on "View on map" tap)
Two action buttons: "Request check-in" and "View on map"
Footer: duration remaining + lock code status
"Settings" link in header goes to TravelShield config for this group
